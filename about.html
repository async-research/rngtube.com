About

Video discovery on YouTube is heavily dependent on what the user would likely want to see. RNGTube tries to make the content discovery on YouTube unique. To make this possible I created a Database of random videos using YoutubeDataAPI. This dynamic site uses the client's randomly generated number to pick a video from the database. Python is used to automate the process of grabbing videos using random words for queries and adding the results to the SQL database. I used Amazon Web Services to host this project via a EC2 Amazon Linux 2 instance.
